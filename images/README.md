# Paper / manuscript figure copies

Key PNGs are written here by **`run_pipeline.py`** (mirrors of `results/plots/`) and by **`export_paper_assets.py`** (sync from `results/plots/` for LaTeX / thesis builds).

- Same filenames as in `results/plots/`, e.g. `mae_by_horizon.png`, `model_performance_1h_ci.png`, `forecast_vs_actual_best_1h.png`, `walk_forward_mae_stability.png`.

In LaTeX you can reference `images/…` locally or the pipeline path `results/plots/…` directly.
To keep GitHub clean, PNG mirrors in this folder are gitignored by default.
